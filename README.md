my\_js

========

My personal tool to manage [Javascript++](https://www.onux.com/jspp/)
projects.

Comman Use Cases:
======

```sh
  my_js js++ my.file.jspp -o my.file.js
  my_js js++ version
  my_js js++ version latest
  my_js install  /to/dir # Creates: /to/dir/JS++
```
